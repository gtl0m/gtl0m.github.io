INNER Quarters/2023-02-25

GIMP

GIMP 足够强大，可以完成我原来在 Photoshop 上进行的 90 % 以上的操作，并且对 avif、webp、ico 等格式都有很好的支持。 更重要的是，当我进行一些简单任务时，它可以在我的电脑上静音运行。所以现在更多打开的是 GIMP 而不是 Photoshop。

googlefonts.cn

Google Fonts 在中国的服务好像寄了。这个我在网站百天的时候就发现了，无奈只能把相关的地方注释掉。但是 font-family 那里还写着 'Ubuntu', sans-serif;。曾经想过把字体也放在 GitHub repo 里，但是 Ubuntu.woff2 有 38 KB…… 恐怕会让本就速度慢的本站雪上加霜。

Fira Code

给 VS code 换上了 Fira Code 字体。全英文的话真的很漂亮，但是这样高大的英文字母如果和汉字放在一起我一时不能接受……应该寻觅一个好的中文字体。
